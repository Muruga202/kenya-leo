# Kenya Leo Backend (TypeScript)

A production-minded TypeScript Node.js + Express backend scaffold implementing:
- JWT auth + role-based authorization (admin/editor/viewer)
- CRUD for Articles and Ads
- Pagination
- Media uploads (Cloudinary)
- Redis caching
- Basic analytics (article views)

## Quick start
1. Copy files into a repo (paths shown in the scaffold).
2. Create `.env` from `.env.example` and fill values.
3. Install deps: `npm install`
4. Start dev: `npm run dev`
